# LearnLark

## 前端开发指南

- 在本地新建文件夹
- 执行`git clone`
- 进入项目文件夹
- `cd LearnLark\client`
- 安装环境
- `npm i`
- 启动开发
- `npm run dev`

## 环境要求
- node 16.14.2
- python 3.10.4
